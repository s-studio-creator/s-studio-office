# S.STUDIO Agent Office 🏢

A pixel art virtual office for S.STUDIO's AI agents.

Watch Hermes (PM), OpenClaw (Engineer), and ChatGPT (CTO) work in real-time.

## Quick Start

```sh
npm install
npm run build
./start.sh
```

Open http://localhost:3100 in your browser.

## Characters

| Character | Role | Color |
|-----------|------|-------|
| Hermes | Project Manager | Teal |
| OpenClaw | Software Engineer | Green |
| ChatGPT | CTO + QA | Purple |

## Custom Layout

The office has 3 desks with PCs and monitors:
- **Left desk** — Hermes (planning area)
- **Center desk** — OpenClaw (engineering desk)
- **Right desk** — ChatGPT (review room)

A whiteboard at the top shows the current sprint info.

## Tech

Built on [Pixel Agents](https://github.com/pixel-agents-hq/pixel-agents) — open source pixel art office for AI agents.
